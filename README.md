# 💱 CurrencyX – конвертер валют

[![GitHub Pages](https://img.shields.io/badge/Демо-currency--converter-ff6b4a?style=flat-square&logo=github)](https://fliperrsv.github.io/currency-converter/)

**CurrencyX** – это современный конвертер валют с актуальными курсами. Поддерживает 5 валют: RUB, USD, EUR, GBP, CNY. Курсы обновляются через API exchangerate-api.com.

🔗 **Живое демо:** [https://fliperrsv.github.io/currency-converter/](https://fliperrsv.github.io/currency-converter/)

---

## 📦 Функционал

- ✅ Конвертация между 5 валютами (RUB, USD, EUR, GBP, CNY)
- ✅ Автоматическое получение актуальных курсов через API
- ✅ Возможность поменять валюты местами
- ✅ Сохранение последних выбранных валют в `localStorage`
- ✅ Тёмная / светлая тема (запоминается)
- ✅ Автообновление курса при смене валют или суммы
- ✅ Адаптивный дизайн для мобильных устройств

---

## 🛠 Технологии

- **HTML5** – семантическая разметка
- **CSS3** – переменные, тёмная тема, адаптив
- **JavaScript (ES6+)** – работа с API, LocalStorage, динамический UI
- **Font Awesome** – иконки
- **exchangerate-api.com** – бесплатный API курсов валют

---

## 🚀 Как запустить локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/fliperrsv/currency-converter.git
   cd currency-converter
```

2. Откройте файл index.html в браузере (или используйте локальный сервер).
3. Всё работает локально, API вызывается напрямую.
